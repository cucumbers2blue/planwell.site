Web App Available

Full planning experience in your browser. Uses the File System Access API to read and write directly to your local vault—no uploads, no sync, no account required. Works in Chrome and Edge. Your markdown files stay exactly where they are.

Open Web App
Desktop App Available

Native macOS app built with Electron. Opens lessons directly in Obsidian, VS Code, Vim, or any editor you configure. Full offline support with direct file system access. Notarized and signed for macOS. Apple silicon native (arm64).

Download for macOS Mac App Store
Obsidian compatible
Homebrew Available

Install and update PlanWell via Homebrew Cask. Same desktop app, managed through your terminal. Auto-updates when you run brew upgrade.

brew install --cask cucumbers2blue/planwell/planwell
Raycast Extension Available

Quick access to your schedule, events, and todos from Raycast. See today's classes, mark todos done, and open lessons directly in PlanWell or Obsidian. Search across all your lesson notes. Keyboard-first workflow for power users.

Install on Raycast
Chrome Extension Available

Toolbar badge shows minutes until your next class. Click to see today's full schedule with color-coded urgency indicators (blue → red as class approaches). Quick glance without opening the full app. Works on Windows, Mac, Linux, Chrome OS.

Note: Chrome shows a "not trusted" warning for new developers. This is normal and will resolve in 2-6 months. PlanWell collects zero data—all processing happens locally on your device.

Install from Chrome Web Store
